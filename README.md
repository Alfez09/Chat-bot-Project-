My_Bot is an interactive chatbot application built with Streamlit and powered by Google’s Gemini generative AI models. It provides a simple web interface for users to ask questions and receive intelligent responses in real time.

Features:

Uses Google Generative AI (Gemini) for natural language responses
Easy-to-use web UI with Streamlit
Secure API key management with dotenv
Modular and extensible codebase
Getting Started:

Clone the repository.
Install dependencies from requirements.txt.
Add your Google API key to the .env file.
Run the app with:
Requirements:

Python 3.13+
Streamlit
google-generativeai
python-dotenv.
